# Simple Was Server

## Dependencies
	* Scala 2.11.11
	* Java 1.8.0 is needed, upper version should be compatible.
	* All other dependencies are managed by Maven.

## Installation
```
	.\install.sh
```
or manually :
```
	mvn clean install
```
